# MIMOFD4precpSense

Beamformer design and measurement-based simulation verification for near-ground precipitation sensing using full-duplex MIMO base stations.

MIMORadar.py: Python package for beamformer design.
simulation.ipynb: measurement-based simulation verification.
LensFD postpro: post-processed partial channel measurement from the LensFD dataset.

Full channel measurement data:
[LensFD dataset](https://renew.rice.edu/dataset-lens.html)  

Reference:
[Near-ground Precipitation Sensing Using Full-duplex MIMO Base Stations](https://ieeexplore.ieee.org/abstract/document/11173435)
